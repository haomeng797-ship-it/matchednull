# Package index

## Build a matched null

Generate synthetic twins that keep the margins and correlations of the
data but contain no cluster structure by construction.

- [`copula_null()`](https://haomeng797-ship-it.github.io/matchednull/reference/copula_null.md)
  : Draw one matched null ("null twin") of a dataset

## Run the test

Compare what a clustering pipeline reports on the real data against what
it reports on the matched nulls.

- [`matched_null_test()`](https://haomeng797-ship-it.github.io/matchednull/reference/matched_null_test.md)
  : Test a cluster count against matched nulls
